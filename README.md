## Начало работы

1. Поддерживаемая версия Node.js: 20+
2. Установите зависимости с помощью

```shell
npm ci
```

3. Запустите проект с помощью 

```shell
npm start
```

4. Дополнительные команды для работы со сборкой

- `npm run build` собирает оптимизированную версию проекта в папке `dist`;
- `npm run preview` запускает сервер с оптимизированной версией;
- `npm run lint`: проверяет стили по правилам stylelint, скрипты по правилам eslint;
